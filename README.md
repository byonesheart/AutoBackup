# AutoBackup v1.1.0

一款 Windows 桌面自动备份应用，支持文件夹和单文件的 ZIP 压缩备份、定时备份、AES 加密、一键恢复等功能。

作者：**byonesheart** | [GitHub](https://github.com/byonesheart/AutoBackup) | [B站](https://space.bilibili.com/481766725)

> 本软件完全免费，无偿分享。

---

## 快速开始

双击 `AutoBackup.exe` 即可运行，无需安装 Python 或其他依赖。

---

## 功能一览

### 备份
- 支持同时备份**多个文件夹和单文件**
- 每个待备份路径单独压缩为 ZIP，存入 `备份位置/文件名备份/` 目录
- 支持 **AES-256 加密**（基于 pyzipper）
- 正在被占用的文件会自动跳过并记录

### 定时备份
- 可设置间隔时间（分钟），到点自动执行
- 开启后显示**预计下次备份时间**
- 关闭软件后重新打开，定时状态自动恢复
- 通过托盘或悬浮窗右键可快速开关定时

### 备份恢复
- 待备份列表和备份记录均可一键恢复
- 恢复时**自动使用备份时的密码**，无需手动输入
- 源文件被删除后仍可恢复（只要备份 ZIP 还在）
- 覆盖式恢复，确认后替换现有文件

### 界面功能
- **待备份列表**：文件名、完整路径、最近备份时间、成功/失败状态
- **状态标记**：绿色圆点 = 启用，红色圆点 = 暂停
- **双击**：文件名打开文件/文件夹，路径打开所在目录，状态标记切换启用/暂停
- **右键菜单**：恢复备份、启用/暂停、移除
- **多选批量操作**：勾选多条后批量移除或切换状态

### 备份记录
- 记录所有备份历史：文件名、路径、时间、是否成功、详情
- 详情显示：是否加密、密码（需开启密码显示）、跳过文件数、失败原因
- 双击路径打开 ZIP 所在目录，双击文件名打开 ZIP
- 可多选移除记录（可选同时删除 ZIP 文件）

### 系统托盘
- 关闭窗口自动最小化到托盘
- 双击托盘图标恢复窗口
- 右键菜单：显示主窗口、立即备份、启动/停止定时、关闭悬浮窗、退出

### 悬浮按钮
- 勾选开启后显示圆形悬浮按钮，始终置顶
- **颜色状态**：蓝色 = 待机，绿色 = 定时运行中，红色 = 有备份失败
- 红色状态在激活主窗口、再次备份成功或重启软件后清除
- **悬停提示**：定时开启时显示下次备份时间
- **左键点击**：立即备份
- **左键长按拖动**：移动位置，靠近屏幕边缘自动吸附为半圆形
- **右键菜单**：立即备份、开关定时、关闭悬浮窗、退出
- 位置自动记忆

### 开机自启
- 勾选后注册到 Windows 启动项，开机最小化运行
- 配合"开机自动备份"可实现开机即备份

### 命令行接口
```bash
# 立即执行一次备份
AutoBackup.exe run

# 查看当前配置
AutoBackup.exe status
```

## 配置文件

程序运行时在 EXE 同目录生成：

| 文件 | 说明 |
|------|------|
| `config.json` | 所有设置（备份路径、间隔、密码、窗口大小等） |
| `backup_log.json` | 备份历史记录 |

---

## 源码运行

```bash
pip install -r requirements.txt
python main.py            # 正常启动
python main.py --minimized  # 最小化启动（托盘常驻）
```

## 技术栈

- **GUI**：PySide6 (Qt 6)
- **压缩/加密**：zipfile / pyzipper (AES)
- **定时**：QTimer
- **打包**：PyInstaller 单文件模式
- **Python**：3.14+

---

## 注意事项

- 加密备份的密码保存在 `backup_log.json` 中，请妥善保管
- 已删除的源文件仍可从备份记录中恢复，前提是 ZIP 未被删除
